Here are the bases of my remote powered by a 9 volt battery. I don't recommend using this build, but I thought I'd share nonetheless.

# Parts list
| Part  |
| ------------- | 
| Arduino Micro  | 
| 5V Regulator |
| Switch |
| Red LED |
| 9V battery |
| SRX582 (reciever) |
| STX882 (transmitter) |
| Opamp for voltage follower |
